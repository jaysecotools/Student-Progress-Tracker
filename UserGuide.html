2. Getting Started

1 Accessing the Application

Open the Vocational Education Progress Tracker in your web browser. The application will load with sample data to help you understand its functionality.

2 Navigation Basics

The application is organized into several main sections accessible via tabs at the top of the screen:

  • Dashboard: Overview of key metrics and recent activity
  • Students: Manage student information and enrollments
  • Courses: Create and manage courses, units, and tasks
  • Reports: Generate various types of reports
  • Analytics: Detailed analysis of unit and task completion
  • Settings: Configure application preferences

3 Understanding Student Status

Students are categorized by their progress status:

Status Description Color Code
On Track Progress is meeting or exceeding expectations Green
Behind Progress is slightly behind schedule Yellow
At Risk Progress is significantly behind schedule Red
Completed All course requirements have been completed Blue

4 Data Persistence

The application stores all data locally in your browser. This means:

  • Your data remains on your computer
  • Data is preserved between browser sessions
  • You can export your data for backup or transfer
Tip: Regularly export your data to prevent loss in case of browser data clearance.

3. Dashboard

The Dashboard provides an overview of your vocational education program with key metrics and visualizations.

Key Metrics

  • Total Students: Number of students in the system
  • Active Courses: Number of courses with enrolled students
  • Completion Rate: Percentage of students who have completed all their enrolled courses
  • At Risk Students: Number of students significantly behind schedule

View Options

The Dashboard offers two view options:

  • Cards View: Displays metrics as summary cards (default)
  • Charts View: Shows visual representations of data including:
    • Student Distribution by Course
    • Average Course Progress
    • Student Status Overview
    • Monthly Activity Trend

Recent Activity

The Recent Activity section shows the latest student activities including task completions, status changes, and other significant events.

Tip: Use the Charts View to quickly identify trends and patterns in student progress.

4. Managing Students

The Students section allows you to manage all student information, enrollments, and progress tracking.

Student List

The student list displays all students in the system with their basic information and enrolled courses. You can:

  • Search: Find students by name using the search box
  • Filter: Filter students by course or status
  • Edit: Modify student information
  • Delete: Remove students from the system
  • View: See detailed student progress

Adding a New Student

  1. Click the "Add New Student" button
  2. Fill in the student's personal information (first name, last name, email, phone)
  3. Select the courses the student is enrolled in
  4. For each selected course, choose the specific units the student will take
  5. For courses with rolling intake, set the student's start date
  6. Click "Save" to add the student

Student Details View

Click "View" on any student to see their detailed progress information:

  • Personal Information: Contact details and overall status
  • Enrolled Courses: List of courses with progress indicators
  • Course Units: Expandable view of units and tasks within each course
  • Task Status: Visual indicators showing completion status of individual tasks

Updating Task Status

In the Student Details view, you can update task completion status by clicking on the circular status indicators next to each task. The status cycles through:

  • Not Started (white)
  • Completed (green)
  • Partial (yellow)
  • Failed (red)
Tip: Use the "Generate Progress Summary" button in the Student Details view to create a printable report of a student's progress.

5. Managing Courses

The Courses section allows you to create and manage vocational courses, including their structure, units, and assessment tasks.

Course List

The course list displays all available courses with information about duration, start dates, and enrollment numbers.

Adding a New Course

  1. Click the "Add New Course" button
  2. Enter course details:
    • Course Code: Unique identifier for the course
    • Course Name: Descriptive name of the course
    • Duration: Length of the course in weeks
    • Start Date: Fixed start date (if not rolling intake)
    • Rolling Intake: Check if students can start at different times
    • Description: Optional course description
  3. Click "Save" to create the course

Managing Course Units

After creating a course, you can add units and assessment tasks:

  1. Click the "Units" button next to a course in the course list
  2. Click "Add Unit" to create a new unit
  3. Enter unit details:
    • Unit Code: Unique identifier for the unit
    • Unit Name: Descriptive name of the unit
    • Description: Optional unit description
    • Order in Course: Position of the unit in the course sequence
  4. Add assessment tasks for the unit using the "Add Task" button
  5. Click "Save" to add the unit
Tip: When setting up courses with rolling intake, remember that each student will have their own start date, allowing for flexible enrollment.

6. Reports

The Reports section allows you to generate various types of reports to analyze student progress and course effectiveness.

Available Report Types

Report Type Description
Completion Report Shows course completion rates and statistics
Progress Report Detailed view of student progress across courses
Attendance Report Activity log filtered by date range
Task Completion Report Detailed breakdown of task completion rates
Custom Report Summary of available data points for custom reporting needs

Generating Reports

  1. Select the report type from the dropdown
  2. Choose a specific course or "All Courses"
  3. Set a date range if applicable
  4. Click "Generate Report"
Tip: Use the Task Completion Report to identify which assessment tasks students are struggling with, allowing for targeted instructional improvements.

7. Analytics

The Analytics section provides advanced tools for analyzing student progress, with a focus on reverse look-up capabilities.

Unit Completion Overview

This section shows completion rates for all units across courses. You can:

  • Filter by course, unit, or task
  • View completion percentages and student counts
  • Click on any unit to see which students have completed it

Task Completion Tracking

This section provides detailed information about specific task completion, showing which students have completed each task and the overall completion rate.

Progress Analytics

This section displays key metrics in an easy-to-read card format:

  • Student Completion Rate
  • Course Completion Rate
  • Unit Completion Rate
  • Task Completion Rate
  • Average Progress
  • At Risk Students Count
Tip: Use the reverse look-up feature to quickly identify students who have completed specific units or tasks, which can be helpful for forming study groups or providing advanced opportunities.

8. Settings

The Settings section allows you to customize the application to match your institution's preferences.

Institution Settings

  • Institution Name: Set your institution's name for display in reports
  • Color Theme: Choose from predefined color schemes or create a custom theme

Data Management

Manage your application data with these options:

  • Export Data: Download all application data as a JSON file for backup or transfer
  • Import Data: Upload previously exported data to restore or transfer information
Warning: Importing data will overwrite all current data. Always export your current data before importing new data to prevent accidental data loss.

9. Troubleshooting

Common Issues and Solutions

Data Not Saving

Problem: Changes to student progress or course information aren't being saved.

Solution: Check if your browser has local storage enabled. The application requires local storage to save data.

Charts Not Displaying

Problem: Charts on the dashboard don't appear or show errors.

Solution: Ensure you have a stable internet connection as the application loads chart libraries from a CDN.

Student Progress Not Calculating Correctly

Problem: Student status or progress percentages seem incorrect.

Solution: Check that all tasks have been properly assigned and that student start dates are correctly set for rolling intake courses.

Application Running Slowly

Problem: The application responds slowly, especially with large datasets.

Solution: Try using the search and filter functions to work with smaller subsets of data. Consider archiving completed courses.

Getting Help

If you encounter issues not covered in this guide:

  1. Check that you're using a supported browser
  2. Clear your browser cache and reload the application
  3. Ensure JavaScript is enabled in your browser
  4. Try exporting your data and reloading the application

Need More Help?
